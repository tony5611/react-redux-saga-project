# React-Redux-Saga project

## Getting Started

1. Clone this repo

`https://github.com/tonyyy93/react-redux-saga-project.git`

2. To run, go to project folder and run

`$ npm install`
or
`$ yarn install` (if you are using yarn)

3. Now start dev server by running -

`$ npm start`
or
`$ yarn start`

4. If the browser does not automatically for any reason visit - http://localhost:3000/

For more create-react-app commands refer `package.json`
